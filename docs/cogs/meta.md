# Meta

## `/ping`

> Returns the bot's latency.   
> **Usage:** `Everyone`   
> **Examples:**     
> `/ping`

## `/serverinfo`

> Returns the server's information.  
> **Usage:** `Everyone`   
> **Examples:**     
> `/serverinfo`

## `/botinfo`

> Returns the bot's information.  
> **Usage:** `Everyone`   
> **Examples:**     
> `/botinfo`

## `/userinfo`

> Returns the user's information. 
> **Usage:** `Everyone`   
> **Examples:**     
> `/userinfo`
> `/userinfo <user>`
